= Frontend

Das Frontend ist das Aushängeschild der Software und ermöglicht den Benutzern die Interaktion mit dem System. Es ist wichtig, dass das Frontend benutzerfreundlich, intuitiv und ansprechend gestaltet ist, um eine positive Benutzererfahrung zu gewährleisten.

Hierbei ist anzumerken, dass das Frontend nicht nur für die Mitarbeiter der SuP Mediengruppe GmbH zugänglich sein soll, sondern auch für die Kunden. Daher muss das Frontend sowohl die internen Geschäftsprozesse als auch die Kundeninteraktionen berücksichtigen und entsprechend gestaltet werden.
